# ETERNA - Time Capsule Application

## Description
ETERNA is a modern time capsule web application that allows users to preserve their memories forever. Create private, shared, or future messages with multimedia content.

## Features
- **User Authentication** (Login/Signup)
- **Private Memories** - Personal capsules visible only to you
- **Shared Memories** - Capsules you can share with others
- **Future Messages** - Lock capsules until a future date
- **Media Support** - Upload images, videos, and audio files
- **Timeline View** - Chronological display of all your memories
- **Statistics Dashboard** - Track your capsule counts
- **Profile Management** - Update your profile and password

## Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: PHP 
- **Database**: MySQL 
- **Libraries**: 
  - SweetAlert2 (Alerts)
  - Font Awesome (Icons)
